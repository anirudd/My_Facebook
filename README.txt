My_Facebook

This project implements a facebook like website as a software in C++. The software offers the user the option of creating an account and then loggin in into the account where he can view his own information. The software has an asthetic user interface that has been implemented by using the header file "graphics.h".

Account information of the users are stored in text files on local storage.

Information about the program logic, functions in the code and the header files used can be found in "anistar-summary.doc" file.

Program File: "ANISTAR1.cpp"

To run the program type the following commands on the linux terminal command window:

g++ ANISTAR1.cpp
./a.out
